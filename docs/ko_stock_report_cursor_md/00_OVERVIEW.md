# 국장/미장 AI 리포트 시스템 개요

목표:
현재 리포트 템플릿을 “데이터 수집 → 분석 → 추론 → AI 투표 → 검수/축약 → UI 출력” 구조로 재설계한다.

## 핵심 구조

```txt
1차 데이터 수집
↓
2차 데이터 정리/분석
↓
3차 AI 추론
↓
4차 AI 투표
↓
5차 리스크 검수/2줄 축약
↓
6차 라벨 + 2줄 코멘트 출력
```

## 핵심 원칙

- 데이터는 많이 수집한다.
- AI는 깊게 분석한다.
- 화면에는 짧게 보여준다.
- 모든 코멘트는 최대 2줄.
- 긴 리포트 본문은 UI에 직접 노출하지 않는다.
- 판단/투표는 유료·상위 모델 사용.
- 단순 2줄 축약/톤 정리는 무료 또는 저비용 모델 사용.

## 최종 UI 출력 요소

```txt
시장 코멘트 2줄
섹터 코멘트 2줄
종목 라벨
라벨 이유 2줄
회사 소개 2줄
AI 투표 결과
```
