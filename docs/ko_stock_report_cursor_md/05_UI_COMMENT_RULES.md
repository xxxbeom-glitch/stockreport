# UI 코멘트 작성 규칙

화면에 노출되는 AI 코멘트는 모두 최대 2줄까지만 허용한다.

## 공통 규칙

```txt
최대 2줄
메모형 문장
입니다/합니다 어투 금지
긴 문장 자동 축약
핵심 근거만 표시
넘치면 말줄임 처리
```

## 1. 마켓 주요 지표 코멘트

내용:
- 코스피/코스닥 흐름
- 환율/금리 영향
- 미국장 영향
- 장전/장중/장후 분위기

예시:
```txt
코스닥 약세와 환율 상승으로 성장주 부담 확대.
대형주는 외국인 수급 중심으로 방어 흐름.
```

## 2. 섹터 유입·유출 코멘트

내용:
- 돈이 들어온 섹터
- 빠져나간 섹터
- 강한 테마
- 관련 종목 흐름

예시:
```txt
반도체 소재주에 HBM 기대감 유입.
삼성전자·SK하이닉스 중심 대형주 강세 동반.
```

## 3. 투자 의견 이유

내용:
- 선택된 라벨의 이유
- 왜 매수 관심인지
- 왜 관망인지
- 왜 지금은 부담인지

예시:
```txt
외국인 수급과 업황 회복 기대가 동시에 붙은 구간.
단기 과열은 있지만 재료 지속성이 더 커 보임.
```

## 4. 회사 소개

내용:
- 기업이 하는 일
- 현재 투자 포인트
- 딱 2줄만 작성

예시:
```txt
메모리 반도체·스마트폰 중심 글로벌 IT 기업.
HBM·AI 반도체·파운드리 회복 기대가 핵심.
```
