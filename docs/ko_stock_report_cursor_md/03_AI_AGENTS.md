# AI 에이전트 역할 정의

국장/미장 리포트 시스템은 4개 역할로 나눈다.

## 1. Report Draft Engine

모델:
`deepseek-v4-flash`

역할:
- 수집 데이터 요약
- 리포트 초안 생성
- 시장/섹터/종목별 핵심 이슈 정리

핵심 질문:
```txt
무슨 일이 있었나?
왜 움직였나?
어떤 데이터가 핵심인가?
```

## 2. DeepSeek Vote Engine

모델:
`deepseek-v4-pro`

역할:
- 데이터 기반 종목 판단
- 상승/하락 원인 추론
- 라벨 후보 제안

핵심 질문:
```txt
이 종목은 데이터상 긍정인가, 중립인가, 부담인가?
```

## 3. Market Pulse Engine

모델:
`grok-4.3`

역할:
- X/커뮤니티 반응 체크
- 언급량 증가 여부 확인
- 과열/루머/밈성 테마 감지
- 시장 심리 보조 판단

핵심 질문:
```txt
시장이 지금 이 종목을 어떻게 반응하고 있나?
```

## 4. Risk Review Engine

모델:
`gemini-3.1-pro-preview`

역할:
- 라벨 판단 리스크 검수
- 과장 표현 제거
- 매수 유도처럼 보이는 문장 완화
- 반대 시나리오 확인

핵심 질문:
```txt
이 판단이 너무 공격적이거나 위험하지 않은가?
```

## 5. Summary Compress Engine

모델:
`gemini-3.1-flash-lite-preview`

역할:
- 최종 문장을 2줄로 축약
- 입니다/합니다 어투 제거
- 메모형 문장으로 정리

핵심 질문:
```txt
이 문장을 2줄 안에 한눈에 보이게 줄일 수 있는가?
```
