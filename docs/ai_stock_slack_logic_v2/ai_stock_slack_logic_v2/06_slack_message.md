# 06. 슬랙 메시지 포맷

## 기본 포맷

```text
[테스트 진입 검토] 종목명

현재가:
예약가 후보:

판단:
왜 지금 볼 만한지 2~3문장으로 설명한다.

진입 관점:
추격매수가 아니라, 눌림이 나오면 1주 테스트 진입을 검토할 만하다는 식으로 작성한다.

주의 조건:
어떤 조건이 깨지면 진입을 취소해야 하는지 작성한다.
```

## 예시 1. 테스트 진입 검토

```text
[테스트 진입 검토] 원익머트리얼즈

현재가: 00,000원
예약가 후보: 00,000원 ~ 00,000원

판단:
반도체 소재주 안에서 거래대금이 다시 붙는 흐름입니다.
아직 급등 추격 구간은 아니고, 수급도 조금씩 개선되고 있습니다.

진입 관점:
1주 테스트 기준으로는 현재가보다 살짝 눌리는 구간에서 진입을 검토해볼 만합니다.

주의 조건:
거래대금이 급격히 줄거나 외국인/기관 수급이 다시 빠지면 오늘은 진입하지 않는 쪽이 좋아 보입니다.
```

## 예시 2. 예약가 제안

```text
[예약가 제안] 한미반도체

현재가: 142,000원
예약가 후보: 138,000원 ~ 140,000원

판단:
HBM 관련 기대감은 유지되고 있고, 장중 거래대금도 살아 있습니다.
다만 현재가는 바로 따라가기엔 부담이 있는 위치입니다.

진입 관점:
급등 추격이 아니라 눌림이 나오는 경우에만 1주 테스트 진입을 검토하는 쪽이 좋아 보입니다.

주의 조건:
140,000원 위로 급하게 올라가거나 거래대금이 빠르게 줄면 오늘은 보류하는 편이 안전합니다.
```

## 예시 3. 관찰 강화

```text
[관찰 강화] 테크윙

현재가: 00,000원
예약가 후보: 00,000원 ~ 00,000원

판단:
반도체 장비주 안에서 오전보다 거래대금이 늘고 있습니다.
섹터 내 다른 장비주와 함께 움직이는 모습이 있어 관찰 강도를 높일 만합니다.

진입 관점:
현재가를 바로 따라가기보다는, 장중 눌림이 나올 때 1주 테스트 진입을 검토할 수 있습니다.

주의 조건:
섹터 동반 흐름이 약해지거나 기관 수급이 빠지면 진입하지 않습니다.
```

## 작성 규칙

- 메시지는 너무 길게 쓰지 않는다.
- 어려운 증권 용어를 과하게 쓰지 않는다.
- 현재가와 예약가 후보는 반드시 표시한다.
- 예약가 후보는 단일 가격이 아니라 범위로 표시한다.
- 비추천 종목은 메시지를 생성하지 않는다.
- 진입 보류, 주의 필요, 판단 애매는 슬랙 발송 금지다.
