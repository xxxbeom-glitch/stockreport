# 02. 슬랙 메시지 발송 목표와 방향

## 메시지 발송 목표

슬랙 메시지는 단순 뉴스 알림이 아니다.

목표는 관심종목 25개 중에서 장중 데이터 흐름상 1주 단위 테스트 진입을 검토해볼 만한 종목만 짧게 알려주는 것이다.

## 핵심 방향

- 추격매수 유도 금지
- 전체 시장 종목 추천 금지
- 관심종목 25개 안에서만 판단
- 비추천, 진입 보류, 판단 애매 종목은 슬랙 발송 금지
- 조건이 맞는 종목이 없으면 아무 메시지도 보내지 않음
- 슬랙은 좋은 기회만 보내는 알림 채널로 사용

## 앱 리포트와 슬랙 메시지 역할 차이

### 앱 리포트

- 장 시작 전/장 종료 후 전체 정리
- 시장 주요 지표
- 관심 섹터 흐름
- 관찰이 필요한 종목
- 부정적 판단도 포함 가능

### 슬랙 메시지

- 장중 변화 감지용
- 실제로 볼 만한 종목만 발송
- 1주 테스트 진입, 예약가 후보, 눌림 진입 가능성 중심
- 부정적 종목은 발송하지 않음

## 슬랙 메시지에 담아야 할 판단

슬랙 메시지는 다음 질문에 답해야 한다.

1. 지금 어떤 종목을 봐야 하는가?
2. 왜 지금 볼 만한가?
3. 현재가 기준으로 바로 추격하면 안 되는가?
4. 눌림이 오면 어느 가격대에서 예약가를 검토할 만한가?
5. 어떤 조건이 깨지면 진입을 취소해야 하는가?

## 권장 표현

- 테스트 진입 검토
- 예약가 후보
- 1주 테스트 기준
- 눌림 구간에서 검토
- 분할 진입 관점
- 거래대금 유지 시 검토
- 수급이 빠지면 진입 보류
