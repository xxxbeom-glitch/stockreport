# 01. 관심 섹터 및 관심종목 기준

## 기본 원칙

- 전체 시장 종목을 자동 추천하지 않는다.
- 아래 관심 섹터와 관심종목 안에서만 분석한다.
- 해운 섹터는 제외한다.
- 현재 1차 관심종목은 5개 섹터, 총 25개 종목이다.

## 관심 섹터

1. 반도체 소재
2. 반도체 부품
3. 반도체 장비
4. 방산·우주
5. 조선·기자재

## 관심종목 목록

### 반도체 소재

| 종목명 | 주요사업 |
|---|---|
| 하나머티리얼즈 | 식각 공정용 SiC 부품 전문 |
| 한솔케미칼 | 반도체용 특수 화학 소재 |
| 원익머트리얼즈 | 반도체용 고순도 특수 가스 |
| 동진쎄미켐 | 감광액(PR) 및 공정 재료 |
| 티씨케이 | SiC 링 및 웨이퍼 부품 |
