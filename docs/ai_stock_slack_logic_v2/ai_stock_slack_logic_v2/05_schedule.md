# 05. 장중 실행 시간대

## 기본 원칙

가격은 장중 계속 변하므로, 슬랙 메시지는 시간대별로 다르게 판단한다.

단, 9시 직후는 변동성이 너무 크므로 바로 판단하지 않는다.

## 추천 실행 시간

### 09:30 - 1차 스캔

장초반 변동성이 어느 정도 줄어든 뒤 첫 판단을 한다.

목적:

- 거래대금이 붙는 종목 확인
- 급등 추격인지 아닌지 확인
- 1차 테스트 진입 검토
- 예약가 후보 제안

### 10:50 - 오전 흐름 유지 확인

장초반에 들어온 돈이 유지되는지 확인한다.

목적:

- 아침 후보가 여전히 유효한지 판단
- 예약가 유지/수정 여부 판단
- 흐름이 약해졌으면 발송하지 않음

### 13:50 - 오후 재유입 확인

점심 이후 다시 돈이 들어오는 종목을 찾는다.

목적:

- 오전보다 흐름이 좋아진 종목 확인
- 오후 재진입 검토 후보 판단
- 1주 테스트 진입 가능성 확인

### 14:50 - 마감 전 최종 확인

마감 전 수급을 확인한다.

목적:

- 당일 진입보다는 내일 관찰 강화 후보 판단
- 그래도 조건이 좋으면 1주 테스트 메시지 발송 가능
- 무리한 마감 전 추격은 방지

## 실행 시간 요약

```text
09:30  장초반 1차 스캔
10:50  오전 흐름 유지 확인
13:50  오후 재유입 확인
14:50  마감 전 최종 확인
```

## 발송 제한

- 각 시간대별 최대 1~3개 종목만 발송
- 조건이 없으면 발송하지 않음
- 같은 종목은 반복 발송하지 않음
- 단, 상태가 크게 좋아지거나 예약가가 의미 있게 바뀐 경우에는 다시 발송 가능
